What is upTee

upTee aims to make the administration simple and fully configurable. It is designed to manage all kind of mods without the problem of missing something.
Also it shows the current status of each server hosted with upTee.

What builds upTee?

upTee depends on many 3rd-party projects. Here's the complete list of them:

  • python - a programming language that you should dive into
    • django - open source web application framework
      • django-admin-tools - pretty admin site
      • django-annoying - a must-have for fixing django annoying things
      • johnny-cache - low-level caching framework
      • django-simple-captcha - simple, yet highly customizable Django application to add captcha images
      • django-pybrowscap - pybrowscap middlware for pybrowscap
      • django-gravatar - easy Gravatar support
      • django-pagination - pagination for dummies
      • south - schema and data migrations
    • celery - distributed task queue
  • nginx - high-performance HTTP server
  • uwsgi - application container server
  • memcached - cache backend
  • rabbitmq - AMQP messaging system, backend for celery
